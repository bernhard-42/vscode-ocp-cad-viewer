# Quickstart experience

The _Quickstart build123d_ and _Quickstart CadQuery_ buttons in the _OCP CAD Viewer_ sidebar set up everything you need in one click:

1. Install the chosen CAD library (`build123d` or `cadquery`) plus `ocp_vscode`, `ocp_tessellate` and `ipykernel` (`jupyter_client`) into the currently selected Python environment, using the commands from `OcpCadViewer.advanced.quickstartCommands`
2. Optionally install the [Jupyter extension for VS Code](https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter)
3. Start the OCP viewer
4. Create a demo file in a temporary folder so you can immediately see a working example
